# 小任务拆分文档（智能匹配 + 积分榜 + 热点图）

## 1. 里程碑规划
- M1：文档与接口契约冻结
- M2：后端初版接口可联调
- M3：前后端联调完成
- M4：优化与验收

## 2. 任务拆分清单

### A. 需求与设计阶段
- A1：整理需求文档与验收标准
- A2：输出系统设计与API契约
- A3：评审并冻结字段命名规范

### B. 后端基础工程
- B1：创建 Spring Boot 工程骨架
- B2：定义统一响应模型
- B3：定义 DTO/VO 与领域模型

### C. 智能解析与匹配
- C1：实现图片解析接口（规则模拟版）
- C2：实现匹配预览接口
- C3：实现匹配打分与可解释原因输出

### D. 统计与可视化接口
- D1：实现积分榜接口
- D2：实现热点图接口
- D3：实现 campus-insights 聚合接口

### E. 前后端联调
- E1：前端替换 mock 为真实接口开关
- E2：联调积分榜页面
- E3：联调热点图页面

### F. 测试与验收
- F1：接口冒烟测试
- F2：异常输入测试
- F3：验收脚本与演示数据固化

## 3. 优先级与依赖
- 必做：B1~B3，C1~C3，D1~D3，E2~E3
- 应做：F1~F2
- 可选：E1（按环境配置切换）
- 依赖关系：A -> B -> (C,D) -> E -> F

## 4. 工时估算（人天）
- A阶段：1.0
- B阶段：1.0
- C阶段：1.5
- D阶段：1.0
- E阶段：1.0
- F阶段：0.5
- 合计：6.0 人天

## 5. 交付物清单
- 需求文档、设计文档、任务拆分文档
- 后端工程初版代码
- 接口示例请求与响应
- 联调说明

## 6. 风险与应对
- 风险：AI服务不可用
  - 应对：保持规则模拟可用，接口契约不变
- 风险：地图资源不可访问
  - 应对：使用本地静态图资源
- 风险：匹配误报高
  - 应对：阈值分层 + 人工复核机制
