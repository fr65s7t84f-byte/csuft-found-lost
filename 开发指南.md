## 项目介绍

### 背景
校园失物招领是高校日常管理的重要环节，传统方式存在信息不对称、流程繁琐、匹配效率低等问题。本系统通过微信小程序+AI技术，实现失物信息智能发布、精准匹配和数据可视化分析。

### 核心特性
- ✅ 微信小程序原生开发，用户体验流畅
- ✅ AI图像识别自动分类和标签提取
- ✅ 智能匹配算法（多维度加权）
- ✅ 实时消息推送通知
- ✅ 数据可视化分析（饼图、热力图）
- ✅ 用户信用评分系统
- ✅ 地图定位功能
- ✅ 评论互动功能

## 核心功能实现

### 1. 智能匹配算法

**算法原理:**

匹配分数由5个维度加权计算得出：

```
总分 = 分类匹配(30%) + 时间匹配(20%) + 地点匹配(20%)
     + 关键词匹配(20%) + AI标签匹配(10%)
```

**详细说明:**

#### 1.1 分类匹配（30分）
```java
private double calculateCategoryScore(String category1, String category2) {
    return category1.equals(category2) ? 1.0 : 0.0;
}
```
- 分类完全一致得满分
- 不一致得0分
- 权重最高，确保基本类别匹配

#### 1.2 时间匹配（20分）
```java
private double calculateTimeScore(LocalDateTime lostTime, LocalDateTime foundTime) {
    long hoursDiff = Math.abs(Duration.between(lostTime, foundTime).toHours());
    long maxHours = 7 * 24;  // 7天

    if (hoursDiff > maxHours) return 0.0;
    return 1.0 - (double) hoursDiff / maxHours;
}
```
- 时间差越小分数越高
- 7天内线性递减
- 超过7天得0分

#### 1.3 地点匹配（20分）
```java
private double calculateLocationScore(Double lat1, Double lng1, Double lat2, Double lng2) {
    double distance = calculateDistance(lat1, lng1, lat2, lng2);

    if (distance <= 1.0) return 1.0;      // 1公里内满分
    else if (distance >= 5.0) return 0.0; // 5公里外0分
    else return 1.0 - (distance - 1.0) / 4.0;
}
```
- 使用Haversine公式计算地球表面两点距离
- 1公里内满分
- 1-5公里线性递减
- 超过5公里得0分

#### 1.4 关键词匹配（20分）
```java
private double calculateKeywordScore(String text1, String text2) {
    Set<String> words1 = tokenize(text1);
    Set<String> words2 = tokenize(text2);

    Set<String> intersection = new HashSet<>(words1);
    intersection.retainAll(words2);

    Set<String> union = new HashSet<>(words1);
    union.addAll(words2);

    return union.isEmpty() ? 0.0 : (double) intersection.size() / union.size();
}
```
- 使用Jaccard相似度算法
- 计算两个文本的词汇交集占比
- 分词采用简单的空格和标点分割

#### 1.5 AI标签匹配（10分）
```java
private double calculateAiTagScore(String tags1, String tags2) {
    Set<String> tagSet1 = new HashSet<>(Arrays.asList(tags1.split(",")));
    Set<String> tagSet2 = new HashSet<>(Arrays.asList(tags2.split(",")));

    Set<String> intersection = new HashSet<>(tagSet1);
    intersection.retainAll(tagSet2);

    Set<String> union = new HashSet<>(tagSet1);
    union.addAll(tagSet2);

    return union.isEmpty() ? 0.0 : (double) intersection.size() / union.size();
}
```
- 计算AI识别标签的交集占比
- 辅助验证物品特征

**匹配阈值:**
- 默认阈值: 60分
- 低于阈值的不推荐
- 可在配置文件中调整

**优化方向:**
1. 引入机器学习模型优化权重
2. 增加用户反馈机制动态调整
3. 考虑物品价值因素
4. 增加图像相似度对比

### 2. AI图像识别

**实现流程:**

```
1. 用户上传图片
   ↓
2. 图片上传至阿里云OSS
   ↓
3. 获取图片URL
   ↓
4. 调用百度AI通用物体识别API
   ↓
5. 解析识别结果
   - 物品名称
   - 颜色
   - 品牌
   - 其他特征
   ↓
6. 自动推荐分类
   ↓
7. 返回前端填充表单
```

**代码实现:**

```java
@Service
public class AiServiceImpl implements AiService {

    @Value("${app.baidu.api-key}")
    private String apiKey;

    @Value("${app.baidu.secret-key}")
    private String secretKey;

    @Override
    public AiRecognizeResult recognizeImage(String imageUrl) {
        // 1. 获取Access Token
        String accessToken = getAccessToken();

        // 2. 调用通用物体识别API
        String apiUrl = "https://aip.baidubce.com/rest/2.0/image-classify/v2/advanced_general";

        Map<String, Object> params = new HashMap<>();
        params.put("url", imageUrl);
        params.put("baike_num", 5);

        String response = HttpUtil.post(apiUrl + "?access_token=" + accessToken, params);

        // 3. 解析结果
        JSONObject json = JSONUtil.parseObj(response);
        JSONArray results = json.getJSONArray("result");

        // 4. 提取关键信息
        List<String> tags = new ArrayList<>();
        String category = "其他";

        for (int i = 0; i < results.size(); i++) {
            JSONObject item = results.getJSONObject(i);
            String keyword = item.getStr("keyword");
            tags.add(keyword);

            // 根据关键词推荐分类
            if (i == 0) {
                category = inferCategory(keyword);
            }
        }

        return new AiRecognizeResult(category, tags);
    }

    private String inferCategory(String keyword) {
        if (keyword.contains("身份证") || keyword.contains("学生证") || keyword.contains("卡")) {
            return "证件";
        } else if (keyword.contains("手机") || keyword.contains("电脑") || keyword.contains("耳机")) {
            return "电子产品";
        } else if (keyword.contains("书") || keyword.contains("笔记本")) {
            return "书籍";
        } else if (keyword.contains("钥匙")) {
            return "钥匙";
        }
        return "其他";
    }
}
```

**注意事项:**
- 百度AI有调用次数限制（免费版每天500次）
- 图片大小不超过4MB
- 支持URL和Base64两种方式
- 识别结果仅供参考，用户可手动修改

### 3. 消息推送

**推送场景:**
1. 有新的匹配推荐
2. 匹配被确认/拒绝
3. 收到评论回复
4. 系统公告

**实现方式:**

```java
@Service
public class MessageServiceImpl implements MessageService {

    @Override
    public void sendMatchNotification(Long userId, Long matchId) {
        // 1. 保存消息到数据库
        Message message = new Message();
        message.setUserId(userId);
        message.setType("match");
        message.setTitle("发现匹配");
        message.setContent("系统为您找到了可能匹配的物品");
        message.setRelatedId(matchId);
        messageMapper.insert(message);

        // 2. 发送微信订阅消息
        sendWechatSubscribeMessage(userId, message);
    }

    private void sendWechatSubscribeMessage(Long userId, Message message) {
        // 获取用户openid
        User user = userMapper.selectById(userId);
        String openid = user.getOpenid();

        // 构造订阅消息
        Map<String, Object> data = new HashMap<>();
        data.put("thing1", message.getTitle());
        data.put("thing2", message.getContent());
        data.put("time3", LocalDateTime.now());

        // 调用微信API发送
        String url = "https://api.weixin.qq.com/cgi-bin/message/subscribe/send?access_token=" + getAccessToken();

        Map<String, Object> params = new HashMap<>();
        params.put("touser", openid);
        params.put("template_id", "your-template-id");
        params.put("page", "pages/match/match");
        params.put("data", data);

        HttpUtil.post(url, JSONUtil.toJsonStr(params));
    }
}
```

## 常见问题

### 1. 数据库连接失败
**问题:** `Communications link failure`

**解决:**
- 检查MySQL是否启动
- 检查防火墙设置
- 检查连接配置是否正确

### 2. JWT Token过期
**问题:** 接口返回401

**解决:**
- Token默认有效期7天
- 前端需要处理Token过期，重新登录

### 3. 图片上传失败
**问题:** 上传图片时报错

**解决:**
- 检查OSS配置是否正确
- 检查文件大小是否超限（10MB）
- 检查文件格式是否支持

### 4. AI识别不准确
**问题:** 识别结果不符合预期
